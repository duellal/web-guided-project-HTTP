# Notes

## HTTP Methods:
| Method    | Meaning                             |
|-----------|-------------------------------------|
| GET       | Read Data                           |
| POST      | Insert Data                         |
| PUT       | Update existing data                |
| PATCH     | Insert new data for existing entry  |
| DELETE    | Delete Data                         |

| C      | R    | U      | D      |
|--------|------|--------|--------|
| Create | Read | Update | Delete |

REST = Representational State Transfer
